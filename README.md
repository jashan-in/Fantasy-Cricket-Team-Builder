## Project Description

This is a full-stack web application where users can create and manage their own fantasy cricket team.

Users can:
- Select a match
- View players based on the selected match
- Build a team of 11 players
- Register their team
- Save team data with authentication

The application uses a front-end built with React and a back-end built with Node.js and Express, with Prisma ORM and a SQL database.

---

## Features

- Multi-page navigation (Matches, Players, My Team, Stats)
- Dynamic player filtering based on match selection
- Team creation with validation (max 11 players)
- User authentication using Clerk
- Persistent data using backend and database
